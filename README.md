# BankSandwich - 소비 습관 진단 웹 애플리케이션

사용자의 소비 습관을 분석하고 맞춤형 재정 조언을 제공하는 웹 애플리케이션입니다.

## 주요 기능

- **21개 설문 질문**: 소비 태도, 생활 습관, 금융 습관 등 7개 카테고리
- **카드형 UI**: 직관적인 선택지 인터페이스
- **소득 입력 슬라이더**: 스크롤 및 직접 입력 지원
- **소비 점수 계산**: 0-100점 기반 등급 평가 (S, A, B, C, D)
- **저축 추천**: 소득과 점수 기반 맞춤형 저축액 제안
- **멍청비용 계산**: 자유롭게 사용 가능한 금액 산출 및 시각화
- **AI 피드백**: 카테고리별 분석 및 개선 조언
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원

## 설치 및 실행

### 프론트엔드

```bash
# 디렉토리 이동
cd D:/banksandwich/frontend

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

개발 서버는 기본적으로 http://localhost:5173 에서 실행됩니다.

## 사용 방법

1. **시작 페이지**: "진단 시작하기" 버튼 클릭
2. **설문 진행**: 21개 질문에 차례대로 답변
   - 1-20번: 객관식 4지선다 (카드 선택)
   - 21번: 월 소득 입력 (슬라이더 또는 직접 입력)
3. **결과 확인**:
   - 소비 점수 (0-100점)
   - 카테고리별 상세 점수
   - AI 분석 및 조언
   - 추천 저축액 및 목표 달성 기간
   - 멍청비용 및 시각화 이미지

## 점수 계산 로직

### 소비 점수
- 각 선택지는 1-4점 (4점이 가장 건전한 소비 습관)
- 총점을 백분율로 환산 (0-100점)
- 등급: S(80+), A(60+), B(40+), C(20+), D(20미만)

### 저축 추천
- 점수에 따라 저축률 결정 (10-30%)
- 월 저축액 = 소득 × 저축률
- 목표 금액 = 1년치 소득

### 멍청비용
- 멍청비용 = 소득 - 필수생활비 - 추천저축액
- 8개 구간별 샌드위치 이미지 매핑
  - 5만원 이하, 5-10만원, 10-20만원, 20-30만원
  - 30-50만원, 50-70만원, 70-100만원, 100만원 이상
